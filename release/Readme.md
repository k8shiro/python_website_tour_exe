## 概要
このソフトウェアは指定されたURLリストを巡回するツールです。起動するとブラウザが起動しURLリストを巡回し続けます。

## 使い方

**起動**

1. urls.jsonを設定

巡回したいurlをurls.jsonに保存

2. settings.jsonを設定

- chrome_path: PCにインストールされているchrome.exeのpathを指定
- scroll_step: ページのスクロール幅

3. main.exeを起動

main.exeを実行

**停止**

起動時に立ち上がったコンソールを閉じる
